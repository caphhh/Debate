# AI辩论赛计时器使用说明

## 文件位置
`debate-timer.html` - 辩论赛计时器主文件

## 辩题信息
- **辩题**: AI产品应先实现用户价值，还是商业价值？
- **正方**: U First（先实现用户价值）
- **反方**: 铁血商团（先实现商业价值）

## 使用方法

### 1. 打开计时器
双击 `debate-timer.html` 文件，在浏览器中打开

### 2. 开始辩论赛
点击首页的"开始辩论赛"按钮

### 3. 计时器控制
- **开始**: 点击"▶ 开始"按钮或按空格键
- **暂停**: 点击"⏸ 暂停"按钮或按空格键
- **重置**: 点击"↺ 重置"按钮或按R键
- **下一环节**: 点击"下一环节 →"按钮或按→键
- **上一环节**: 点击"← 上一环节"按钮或按←键
- **全屏**: 点击"⛶ 全屏"按钮或按F键
- **静音**: 点击右下角🔊按钮或按M键

### 4. 编辑时间
点击"✎ 编辑时间"按钮可以修改各个环节的时长

### 5. 环节导航
右侧显示所有环节列表，点击可跳转到指定环节

## 声音提示
- **剩余30秒**: 响起警示音（3声）
- **时间到**: 响起5秒倒计时音效
- **可静音**: 点击右上角🔊按钮关闭声音

## 特色功能

## 注意事项
- 首次打开浏览器可能需要允许音频自动播放
- 建议在辩论前预先测试所有功能
- 可以提前编辑好各环节时间
- 使用全屏模式获得最佳展示效果
