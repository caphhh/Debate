# 🎤 AI辩论赛计时器 - 在线网站

## 快速使用

### 方法一：本地服务器（推荐用于局域网分享）

1. 双击运行 `start-server.sh` 文件
2. 在浏览器中打开 `http://localhost:8080`
3. 分享给同一网络下的其他人，告诉他们你的IP地址 + `:8080`

### 方法二：部署到免费云平台

#### 选项1：Netlify（最简单）
1. 访问 https://www.netlify.com/
2. 注册/登录账号
3. 将整个 `debate-timer-site` 文件夹拖拽到部署区域
4. 立即获得一个可访问的网址，如 `https://xxx.netlify.app`

#### 选项2：GitHub Pages
1. 在GitHub上创建新仓库
2. 上传此文件夹内容
3. 在仓库 Settings → Pages 中选择分支
4. 获得网址如 `https://username.github.io/repo-name`

#### 选项3：Vercel
1. 访问 https://vercel.com/
2. 导入文件夹
3. 自动部署并获得网址

## 功能特点

- ✅ 完整的辩论赛流程（17个环节）
- ✅ 实时倒计时 + 30秒警示音 + 5秒倒计时提示
- ✅ 1v1对辩和自由辩论独立计时
- ✅ 可编辑辩题、队名、观点
- ✅ 设置自动保存
- ✅ 全屏显示模式
- ✅ 响应式设计，支持手机/平板

## 文件说明

- `index.html` - 主程序文件
- `start-server.sh` - 本地服务器启动脚本
- `README.md` - 使用说明

## 注意事项

- 首次使用时，浏览器可能会阻止自动播放音频，需要用户交互一次
- 建议在辩论前预先测试所有功能
- 使用全屏模式（F键）获得最佳展示效果
