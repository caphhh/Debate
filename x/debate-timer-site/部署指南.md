# 🚀 一键部署指南

## 最简单的方法：Netlify Drop（30秒搞定）

### 步骤：

1. **打开网站**
   在浏览器中访问：https://app.netlify.com/drop

2. **拖拽文件夹**
   将整个 `debate-timer-site` 文件夹拖拽到页面上

3. **获得网址**
   几秒钟后，你会得到一个类似这样的网址：
   ```
   https://xxx-xxx-xxx.netlify.app
   ```

4. **分享给别人**
   把这个链接发给任何人，他们都可以打开使用！

---

## 备选方案：GitHub Pages

1. 创建一个新的GitHub仓库
2. 上传 `debate-timer-site` 文件夹中的所有文件
3. 进入仓库 Settings → Pages
4. 选择 main 分支，保存
5. 获得网址：`https://你的用户名.github.io/仓库名`

---

## 备选方案：Vercel

1. 访问 https://vercel.com/new
2. 导入文件夹
3. 自动部署，获得网址

---

## 验证网站是否正常工作

部署后，用另一个浏览器或手机打开网址，测试：
- [ ] 首页是否正常显示
- [ ] 能否正常开始辩论赛
- [ ] 计时功能是否正常
- [ ] 提示音是否能播放
- [ ] 设置能否保存

---

## 常见问题

**Q: 提示音无法播放？**
A: 首次使用时需要在页面上点击一次（如点击"开始辩论赛"），浏览器才允许播放音频。

**Q: 如何修改默认辩题和队名？**
A: 点击右上角的 ⚙️ 设置按钮即可修改，设置会自动保存。

**Q: 可以在手机上使用吗？**
A: 可以！网站已经做了响应式设计，手机、平板都能正常使用。
